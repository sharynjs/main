## Installation

```sh
npm i --save-dev @sharyn/tsconfig
# or
yarn add --dev @sharyn/tsconfig
```

## Usage

For client bundles, in `tsconfig.json`, use:

```json
{
  "extends": "@sharyn/tsconfig"
}
```

For server bundkes, in `tsconfig.json`, use:

```json
{
  "extends": "@sharyn/tsconfig/server"
}
```
